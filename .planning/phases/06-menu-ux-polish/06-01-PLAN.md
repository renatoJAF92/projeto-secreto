# Plan 06-01: Audio Infrastructure

## Goal
Criar o sistema de audio buses (Music / SFX) separado do Master bus, adicionar métodos de controle de volume no AudioManager, e persistir os valores de volume no SaveManager.

## Context
- `autoloads/audio_manager.gd`: `_music_player` no bus "Master"; SFX players também em "Master". Sem métodos de volume.
- Audio buses em Godot 4 são configurados via `AudioBusLayout` resource (arquivo `.tres`) referenciado em `project.godot` como `audio/buses/default_bus_layout`.
- `autoloads/save_manager.gd`: `current_save` Dictionary, método `save()` persiste em disco.

## Tasks

### T-06-01-01: Criar default_bus_layout.tres com 3 buses
**File:** `assets/audio/default_bus_layout.tres`

Criar o arquivo de bus layout com 3 buses: Master (idx 0), Music (idx 1), SFX (idx 2).

```gdresource
[gd_resource type="AudioBusLayout" format=3]

[resource]
bus_count = 3
bus/0/name = &"Master"
bus/0/solo = false
bus/0/mute = false
bus/0/bypass_fx = false
bus/0/volume_db = 0.0
bus/0/send = &""
bus/1/name = &"Music"
bus/1/solo = false
bus/1/mute = false
bus/1/bypass_fx = false
bus/1/volume_db = 0.0
bus/1/send = &"Master"
bus/2/name = &"SFX"
bus/2/solo = false
bus/2/mute = false
bus/2/bypass_fx = false
bus/2/volume_db = 0.0
bus/2/send = &"Master"
```

### T-06-01-02: Referenciar bus layout em project.godot
**File:** `project.godot`

Adicionar linha na seção `[audio]`:
```ini
[audio]
buses/default_bus_layout="res://assets/audio/default_bus_layout.tres"
```

Se a seção `[audio]` já existir, adicionar apenas a linha `buses/default_bus_layout`.

### T-06-01-03: Atualizar AudioManager para usar buses corretos
**File:** `autoloads/audio_manager.gd`

1. Mudar `_music_player.bus = "Master"` → `_music_player.bus = "Music"`
2. Em `register_sfx()`: mudar `player.bus = "Master"` → `player.bus = "SFX"`
3. Adicionar constante de índices de bus:
```gdscript
const _MUSIC_BUS := 1  # índice do bus "Music" no AudioBusLayout
const _SFX_BUS := 2    # índice do bus "SFX" no AudioBusLayout
```
4. Adicionar métodos de volume:
```gdscript
func set_music_volume(linear: float) -> void:
    AudioServer.set_bus_volume_db(_MUSIC_BUS, linear_to_db(linear) if linear > 0.0 else -80.0)

func set_sfx_volume(linear: float) -> void:
    AudioServer.set_bus_volume_db(_SFX_BUS, linear_to_db(linear) if linear > 0.0 else -80.0)

func get_music_volume() -> float:
    var db := AudioServer.get_bus_volume_db(_MUSIC_BUS)
    return 0.0 if db <= -79.0 else db_to_linear(db)

func get_sfx_volume() -> float:
    var db := AudioServer.get_bus_volume_db(_SFX_BUS)
    return 0.0 if db <= -79.0 else db_to_linear(db)
```
5. Em `_ready()`, após criar `_music_player`, carregar volumes salvos:
```gdscript
func _ready() -> void:
    _music_player = AudioStreamPlayer.new()
    _music_player.bus = "Music"
    add_child(_music_player)
    _apply_saved_volumes()
    # ... sfx registration existente ...

func _apply_saved_volumes() -> void:
    var music_vol: float = SaveManager.current_save.get("music_volume", 0.8)
    var sfx_vol: float = SaveManager.current_save.get("sfx_volume", 1.0)
    set_music_volume(music_vol)
    set_sfx_volume(sfx_vol)
```

### T-06-01-04: Adicionar campos de volume ao SaveManager
**File:** `autoloads/save_manager.gd`

No método `new_game()` (ou onde o save é inicializado com defaults), garantir que os campos de volume existam com valores default:
```gdscript
current_save["music_volume"] = 0.8
current_save["sfx_volume"] = 1.0
```

Verificar se `new_game()` já inicializa `current_save` com um dicionário fixo — se sim, adicionar os campos. Se usa `.get()` defensivo, os defaults já funcionam sem mudança (AudioManager._apply_saved_volumes usa `.get(..., default)`).

### T-06-01-05: Verificar que SFX existentes ainda tocam
Executar `grep -r "AudioManager.play_sfx" scenes/` e confirmar que nenhuma chamada foi quebrada.
Os SFX players serão no bus "SFX" agora — volumes diferentes do Master não afetam o comportamento, apenas o controle separado funciona.

## Acceptance Criteria
- [ ] `assets/audio/default_bus_layout.tres` existe com 3 buses (Master, Music, SFX)
- [ ] `project.godot` referencia o bus layout
- [ ] `AudioManager.set_music_volume(0.5)` reduz volume da música em 50% (testável em tempo de execução)
- [ ] `AudioManager.set_sfx_volume(0.0)` silencia SFX sem afetar música
- [ ] Volumes default (0.8 música, 1.0 SFX) aplicados ao iniciar o jogo
- [ ] Nenhum SFX existente quebrado (stomp, jump, checkpoint etc. ainda tocam)

## Notes
- `linear_to_db(0.0)` retorna `-inf` em GDScript — guardar com o check `if linear > 0.0` ou o jogo crasha ao setar silêncio.
- Bus indices são fixos no layout file — se alguém editar o `.tres` manualmente e reordenar os buses, os índices precisam ser atualizados.
